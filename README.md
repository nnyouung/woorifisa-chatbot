## Getting Started

### 1) Install
```npm install```

### 2) Create .env
```
CLOVA_CHATBOT_INVOKE_URL=...
CLOVA_CHATBOT_SECRET=...
```

### 3) Run
```npm start```

### 4) Open
http://localhost:3000
